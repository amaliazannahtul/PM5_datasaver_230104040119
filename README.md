Prodi : Teknologi Informasi  

---

## Deskripsi
Praktikum ini bertujuan untuk memahami penyimpanan data
pada aplikasi Android menggunakan:
- SharedPreferences
- File Handling
dengan pendekatan Jetpack Compose.

---

## Fitur yang Dipelajari
- Menyimpan data sederhana menggunakan SharedPreferences
- Menyimpan dan membaca data menggunakan File Handling
- Penggunaan Repository
- Struktur project Android dengan Jetpack Compose

---

## Tools & Teknologi
- Kotlin
- Jetpack Compose
- GitHub

## Penjelasan Singkat
- PreferencesRepository.kt digunakan untuk mengelola SharedPreferences
- FilesRepository.kt digunakan untuk pengelolaan file
- Folder ui berisi tampilan aplikasi
- Folder navigation mengatur perpindahan antar screen
